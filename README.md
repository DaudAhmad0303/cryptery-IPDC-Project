## Welcome to cryptery

this is a C algorithm to decrypt w/ brute-force hashes

## Getting Started

Compiling with Clang

```
clang cryptery.c -o cryptery -lpthread && ./cryptery __HASH__
```

## Usage

```
./cryptery __HASH__
```
