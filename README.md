## Welcome to cryptery

this is a C algorithm to decrypt w/ brute-force hashes

## Getting Started

Compiling with Clang

```
mpicc cryptery.c -o cryptery -std=c11 -lcrypto
```

## Usage

```
mpirun -np 4 ./cryptery
```
