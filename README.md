## Welcome to cryptery

this is a C algorithm to decrypt w/ brute-force hashes

This branch uses OpenMP to paralelize

## Getting Started

Compiling with gcc (Very very sad about that)

```
gcc cryptery.c -o cryptery -fopenmp -std=c11 -lpthread -lcrypto
```

## Usage

```
./cryptery __HASH__
```
